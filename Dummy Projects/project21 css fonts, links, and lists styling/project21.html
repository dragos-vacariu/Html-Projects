The font family of a text is set with the font-family property. The font-family property should hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font, and so on.

The font-style property is mostly used to specify italic text, but can also hold several values:

  • normal - The text is shown normally
  • italic - The text is shown in italics
  • oblique - The text is "leaning" (oblique is very similar to italic, but less supported)

The font-size property sets the size of the text. It can be set with pixels (px) or em (1em = 16px)

The font-weight property specifies the weight of a font (normal or bold.)

The font-variant property specifies whether or not a text should be displayed in a small-caps font.

CSS Links:

Links can be styled with any CSS property (e.g. color, font-family, background, etc.)

A link has 4 states:

  • a:link - a normal, unvisited link
  • a:visited - a link the user has visited
  • a:hover - a link when the user mouses over it
  • a:active - a link the moment it is clicked

Any of those states can be styled using CSS properties.
