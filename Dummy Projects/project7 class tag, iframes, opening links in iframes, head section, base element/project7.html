Absolute filepath for an image: <'img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">

Relative filepath for an image: <'img src="./images/picture.jpg" alt="Mountain">

An absolute filepath is the full URL to an internet file or a local file, whereas a relative filepath points to a file relatively placed to the current location in the directory or on the web.

It is a best practice to use relative file paths (if possible). When using relative filepaths, web pages will not be bound to your current base URL. All links will work on your own computer (localhost) as well as on your current public domain and your future public domains.
