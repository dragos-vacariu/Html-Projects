CSS Align:

To horizontally center a block element (like <'div> or image), use margin: auto;

The property text-align: center can be used to center the text.

Other alignments can be achieved using float property and position:absolute property.

CSS Selectors:

There are four different combinators in CSS3:

  • descendant selector (space - div p {background-color: yellow;} )
  • child selector (> - div > p { background-color: yellow; })
  • adjacent sibling selector (+ - div + p { background-color: yellow;} )
  • general sibling selector (~ - div ~ p {background-color: yellow;} )

CSS Pseudo-classes:

A pseudo-class is a class used to define a special state of an element.

Pseudo-elements:

For example, it can be used to:
  • Style the first letter, or line, of an element
  • Insert content before, or after, the content of an element

These properties can be applied to a pseudo-element:
  • font properties
  • color properties
  • background properties
  • word-spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transform
  • line-height
  • clear
