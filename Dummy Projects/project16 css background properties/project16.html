CSS selectors are used to "find" (or select) HTML elements based on their element name, id, class, attribute, and more.

The id selector uses the id attribute of an HTML element to select a specific element. Example: #name{color:#000000}, where name is the element ID(as seen in the example # sign is required in front of the ID.)

The class selector selects elements with a specific class attribute. Example: h.name {color: #000000} , where name is the class_name of the element.

CSS comments can be written using this syntax /* comment here */ just like in other programming languages.

Multiple Style Sheets - if some properties have been defined for the same selector (element) in different style sheets, the value from the last read style sheet will be used.

CSS does not have attributes, but only properties, so background-color is a property not an attribute. Also font-family, and font-size, and so on.

To set the background-image using CSS the syntax is: body{background-image:url("image.jpg");}
